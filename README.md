# Akash & Pranshi Wedding Invitation

A small game-style static site for a wedding invitation that is ready to publish on GitHub Pages.

## Files
- `index.html` – landing page and invitation game
- `styles.css` – page styling
- `script.js` – interactive game logic

## Publish with GitHub Pages
1. Push this repository to GitHub.
2. In the repository settings, enable GitHub Pages from the `main` branch.
3. The site will be available at `https://<your-username>.github.io/<repository-name>/`.
